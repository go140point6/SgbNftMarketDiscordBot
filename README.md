## Run with Docker
Edit .env file (Keep database default values if you want to use an embedded database)

## Run
Edit src/main/application.properties then
```sh
mvn spring-boot:run
```

```sh
docker-compose up
```

## Test
```sh
mvn test
```

## Build
```sh
mvn clean install
```
